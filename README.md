# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/AnujGangwar01/Portfolio.git
```

## Navigate into the project

```bash
cd Portfolio
```

## Install dependencies

```bash
npm install
```

## Start development server

```bash
npm run dev
```

Visit:

```
http://localhost:5173
```
